# 📈 伸動保健室｜營收管理與預測系統 (ST-Clinic-Pro)

這是一個專為 **伸動保健室** 打造的數位化營運後台。整合了每日總帳、客戶消費明細管理，以及 AI 線性迴歸業績預測功能，幫助團隊精準掌握經營動態。

## ✨ 核心功能
* **業績預測儀表板**：利用線性迴歸分析，根據當前數據預測月底的入金與消化總額。
* **團隊績效追蹤**：支援成員的個別業績與總覽切換。
* **客戶消費管理**：記錄客戶名稱、訂單來源、入金方式及購買品項。
* **自動統計分析**：
    * **來店人次**：自動區分新客體驗與舊客續課/扣點。
    * **金流統計**：現金、街口、Line Pay、轉帳、刷卡等多元支付匯總。
* **資料防彈機制**：
    * **自動存檔**：資料即時儲存於瀏覽器 localStorage，不怕斷網或關閉視窗。
    * **備份匯出**：一鍵下載 CSV 檔案，方便進行離線備份與二開發。
    * **批量匯入**：支援 CSV 檔案上傳，快速轉移舊有資料。

## 🛠️ 技術工具
* **React**：前端開發框架。
* **Recharts**：專業數據圖表可視化。
* **Tailwind CSS**：現代化風格介面設計。
* **Lucide React**：精美功能圖示。

## 🚀 快速開始
本系統已部署於 Vercel，點擊專案連結即可開始使用。
* **資料儲存提醒**：資料儲存在您的個人電腦瀏覽器中，若更換設備或清除瀏覽器快取，請先點擊「下載備份」確保資料安全。
